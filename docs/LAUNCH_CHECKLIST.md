# Launch Checklist (Reddit + Portfolio)

## 1) Demo readiness

- [ ] Open demo in Incognito and verify no broken assets.
- [ ] Scroll through full story, gallery open/close, RSVP submit state.
- [ ] Verify mobile view (Chrome device emulation).

## 2) Repo readiness

- [ ] README includes demo + source links.
- [ ] `.env` is not committed.
- [ ] Placeholder content only (no personal names/addresses).

## 3) Post assets

- [ ] 1 hero screenshot
- [ ] 1 gallery interaction screenshot
- [ ] 1 RSVP/UX screenshot
- [ ] Optional 20-40s screen recording (best conversion)

## 4) Reddit publishing

- [ ] Use title/body from `docs/REDDIT_POST.md`.
- [ ] Post in one community first, iterate from feedback.
- [ ] Reply to comments in first 30-60 minutes.

## 5) Lead capture

- [ ] Add contact CTA in profile/README (Telegram/Email).
- [ ] Keep a short offer ready (see `docs/CLIENT_OFFER_TEMPLATE.md`).
